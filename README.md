# VHL-Solver-Extension
VHL Solver --- Extension to solve VHL

INSTRUCTIONS
------------
DOWNLOADING
Download the code
Go to chrome://extensions
Open the button that says "Load unpacked"
Then, open the folder that has the subfolder - "VHL Solver" in that
Upload that folder into "Load unpacked", and click apply
Then go to VHL & click the icon or whatever

RUNNING
Go to OpenRouter.com
Make an account
Make an API key from that account
Go to the extension menu & go to the OpenRouter tab
Paste in the key and click solve
